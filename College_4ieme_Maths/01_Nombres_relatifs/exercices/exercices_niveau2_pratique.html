Exercice 1 – Somme multiple

(-7) + (+12) + (-5)

Méthode : Regrouper par signes : (+12) et (-7-5).

Solution : (+12) + (-12) = 0

Exercice 2 – Soustraction

(-15) - (-9)

Méthode : Transformer : -(-9) = +9.

Solution : (-15) + (+9) = -(15-9) = -6

Exercice 3 – Budget familial

Compte : +250€. Dépenses : -180€, -95€. Recette : +120€. Solde final ?

Méthode : (+250) + (-180) + (-95) + (+120).

Solution : Recettes 370€, dépenses 275€ → +95€

Exercice 4 – Séquence d'opérations

(+8) - (+3) + (-12) - (-7)

Méthode : Transformer : (+8) + (-3) + (-12) + (+7).

Solution : Positifs : 8+7=15, Négatifs : 3+12=15 → 0

Exercice 5 – Variation de température

Lundi : -8°C. Mardi : monte de 12°C. Mercredi : baisse de 15°C. Température mercredi ?

Méthode : (-8) + (+12) + (-15).

Solution : Lundi→Mardi : +4°C, puis -15°C → -11°C
