Fiche Synthèse – Nombres relatifs

Cours

📖 Règles Essentielles

➕RÈGLE 1 : Même signe

Additionner les valeurs et garder le signe commun

Exemples :
• (-3) + (-5) = -8 (3 + 5 = 8, signe -)
• (+6) + (+4) = +10 (6 + 4 = 10, signe +)

➖RÈGLE 2 : Signes différents

Soustraire la plus petite de la plus grande, garder le signe du « plus fort »

Exemples :
• (-7) + (+4) = -3 (7 - 4 = 3, signe de -7)
• (+9) + (-2) = +7 (9 - 2 = 7, signe de +9)

🔍RÈGLE 3 : Contrôle

Vérifier avec un contexte concret

Astuces :
• Température : -5°C + 8°C = +3°C ✓
• Argent : -20€ + 15€ = -5€ (découvert) ✓

Erreur : (-4) + (-3) = -1 (faire 4-3)

✓ Correct : Même signe → 4+3 = 7 → -7

🚫 Piège 2 : Mauvais signe final

Erreur : (+2) + (-8) = +6 (garder le +)

✓ Correct : Le « plus fort » est -8 → -6
